# 🚀 Intelligent Sensor Network for Smarter Border Security

## 📌 Overview
This project is a Machine Learning-based system designed to improve border security by analyzing sensor data and detecting suspicious activities.

---

## 🧠 Model Used
- Random Forest Classifier

---

## ⚙️ Features
- Sensor data simulation
- Human activity detection
- Classification of normal vs suspicious activity
- Performance evaluation using accuracy and confusion matrix

---

## 🛠️ Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

---

## 📊 How It Works
1. Generate sensor data
2. Train machine learning model
3. Predict suspicious activities
4. Evaluate results

---

## ▶️ How to Run
1. Open the `.ipynb` file in Jupyter Notebook
2. Run all cells
3. View results and graphs
